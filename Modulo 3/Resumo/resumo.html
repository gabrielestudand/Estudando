2-O que é Git e GitHub?

git- gerenciador de versões , para salvar várias versões do mesmo projeto caso eu não tenha gostado de alguma alteração

github - guarda meus projetos online e na nuvem caso aconteça algo ao meu computador e serve como um servidor para apresentar meus projetos a empresas , cliente ou amigo

git - repositorio local
github- repositorio remoto
push : jogar arquivos no repositorio remoto
