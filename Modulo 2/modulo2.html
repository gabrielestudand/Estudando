Aula 36

Variaveis em html css

Para criar uma variavel primeiro vou criar uma pseudo classe chamada :root , então tudo o que eu definir nessa pseudo classe vai servir para o documento todo

Todas as variaveis começam com = --(variavel)

Exemplo : 

             
                /*
                 #04BF8A;
                 #025940;   (LISTA DE CORES PARA VARIAVEI)
                 #03A64A;
                 #defff4;
                *\
                :root{   ( COMANDO DA VARIÁVEL ,NOME DA VARIAVEL E  CODIGOS JUNTOS)
                    --cor0:  #04BF8A;
                    --cor1:  #025940;
                }
                body{
                    background-color: var(--cor0); (EXEMPLO EM ALGO)
                }
